The Pivot

From Pixels to Code

For over five years I shaped brands, designed marketing campaigns, and built social-media identities for companies across Turkey, Turkmenistan, and Malaysia. I learned that great visuals are only half the story — the other half lives in the code that delivers them.

Now, while pursuing a B.A.S. in Web Application & Cloud Development at Edmonds College, I'm channeling that same creative precision into front-end engineering — writing semantic HTML, modular CSS, and purposeful JavaScript.

My goal: to sit at the intersection of design systems and clean code, translating ideas into performant, accessible user experiences.
